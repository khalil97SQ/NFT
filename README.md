# NFT Project

This project is a **static NFT image gallery** created as a **training exercise to learn React and Vite**.  
The goal of this project is to **practice displaying images using a slider**.

## Project Preview

![NFT Screen](./src/assets/NFT%20Screen.jpeg)

## Technologies Used

- **React**: for building interactive user interfaces
- **Vite**: as a fast build and development tool
- **Splide**: to display images in a responsive slider
- **CSS**: for styling and making the gallery look appealing

## Notes

- Images are displayed in an interactive slider, allowing easy navigation
- This project is for **training purposes only**

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```
   ```
   npm run dev
   ```
